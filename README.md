### 🏆 **1. AuctionXI -- Real-Time IPL Auction Simulation Platform**
**Tech Stack:** `Node.js` • `Express.js` • `React` • `Socket.io` • `PostgreSQL` • `Tailwind CSS`  
**Repository:** [🔗 GitHub](https://github.com/duveen2546R/auction-x1) | **Live Demo:** [🎮 Play Now](https://auction-x1-frontend.onrender.com/)

> Built a **real-time multiplayer cricket auction platform** for IPL-style team building with live bidding and synchronized auction state.

**Key Features:**
- ⚡ Live bidding with real-time state synchronization using **Socket.io**
- 🎯 Auction logic for player pools, budget constraints, role limits, and squad validation
- 💰 Scoring system with dynamic team rankings
- 🎨 Responsive React frontend with Vite & Tailwind CSS
- 🗄️ Persistent data storage with Node.js + PostgreSQL backend
- ☁️ Deployed on Render for live gameplay

**Highlights:**
- Supports multi-player concurrent bidding
- Real-time player statistics and team compositions
- Dynamic auction flow management
- Mobile-responsive UI

- Support for multiple prediction algorithms
